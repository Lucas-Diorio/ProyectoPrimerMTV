Web de gestión de stock

Nombre Proyecto: ProyectoPrimerMTV
Nombre App: Stock

Contiene HTML padre que contiene toda la estructura de los demás templates.
Contiene los modelos: Productos, Entradas y Salidas.
Contiene Formulario para ingresar datos a cada modelo/tabla de la base de datos.
Contiene un formulario de búsqueda de productos.



Links: 

    Link de inicio: http://127.0.0.1:8000/Stock/
    Link de productos: http://127.0.0.1:8000/Stock/productos/ (Página vacía, solo informativa).
    Link de entradas: http://127.0.0.1:8000/Stock/entradas/ (Formulario para ingresar datos de entradas en Stock).
    Link de salidas: http://127.0.0.1:8000/Stock/entradas/ (Formulario para ingresar datos de salidas de Stock).
    Link de datos de productos: http://127.0.0.1:8000/Stock/productoFormulario/ (Formulario para ingresar datos de productos).
    Link de datos de productos: http://127.0.0.1:8000/Stock/resultadosBusqueda/ (Formulario de búsqueda de productos por nroparte).
